# 淫话……粗话……脏话小说国产淫语对白粗口刺激

嘴角露出笑意，目光柔和，能以人类的身份跟他相处，就算只有一个月，也知足了。
　　六尾狐则是跟在后面，眼巴巴的望着自己的姐姐被那个凡人带走。
　　此时，方圆万里内，不管是修仙者还是大妖全都轰动起来。
　　天空中一道道流光窜射而过，以极快的速度，从四面八方向着刚刚渡劫的地方涌来。
　　乾龙仙朝中。
　　洛诗雨看着雷电的方向，脸上带着一丝惊叹，低声道：“想不到居然真的会有妖选择化形，百年未见了吧。”
　　“这妖怪能够放弃千年修行，冒着天大的危机选择化形，确实非常难得。”钟秀点了点头道。
　　妖怪化形，九死一生，往往是个妖怪能幸存下来一个就不错了。
　　这得有着何等的气魄才会选择化形啊。
　　“娘，你看那里的方向，距离高人的住处好像不远。”洛诗雨微微蹙眉，打量着天际。
　　钟秀点了点头，“确实很近，也不知道会不会打扰到高人清修。”
　　“娘，这么大的事情，肯定会吸引很多人过来，我得过去，省的有不开眼的人冒犯到高人。”洛诗雨凝重道。
　　正说话间，圣皇也是从外面急匆匆的赶来。
　　“诗雨，我跟你一起过去！高人的清修之地不容许任何人打扰！”圣皇语气坚决，不容置喙，“这是整个乾龙仙朝的意志！”
　　洛诗雨和钟秀都愣住了。
　　“爹，你不去妖怪渡劫的地方看看？”洛诗雨问道。
　　现在无疑是化形妖怪最为脆弱的时候，一旦能够找到，那将会是一笔无比巨大的财富。
　　圣皇摇了摇头，“妖怪化形虽然少见，但跟高人相比，算得了什么？高人能够隐居在我乾龙仙朝，这是我们整个仙朝的荣幸，若是因为被人打扰而离开，那这个损失可就大了！”
　　洛诗雨和钟秀深以为然的点了点头。
　　高人若是一怒之下离开了乾龙仙朝去了其他地方隐居，那是整个仙朝的损失！
　　“走，我们一起过去！”钟秀也是连忙道。
　　他们不敢耽搁。
　　当即，乾龙仙朝地位最高的三人没人惊动任何人，悄摸摸的一同出发，向着李念凡的住处而去。
